barcoding
UPC stands for Universal Product Code, which was the first system of bar code labelling to be widely adopted.
UPC led to the adoption of the EAN code format, (a format similar to UPC) 
EAN-8 and EAN-13 symbols in conjunction with 12 numerical digit UPC will ensure global conformity.
 The first digit L indicates a particular number system to be used by the following digits.
 The last digit R is an error detecting check digit that allows some errors in scanning or manual entry to be detected.
1  23456 78999  9

In short, the UPC is a 12-digit unique code for your product represented by scannable bars.
